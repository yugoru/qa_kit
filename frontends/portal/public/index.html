QK
QA Kit
Hands-on playground for mid-level QA
Практический стенд для тестировщиков: REST, gRPC, GraphQL, брокеры сообщений, логи, метрики и многое другое.
Как пользоваться
Запусти docker compose up, открой этот портал и переходи по ссылкам к нужным сервисам. Каждый блок курса использует один или несколько сервисов снизу. Для перехвата трафика и работы с DevTools используй Playground.
